# TelePrompter Pro

발표자·영상 제작자를 위한 전문 브라우저 텔레프롬프터입니다.

## 기능

- **부드러운 자동 스크롤** — 속도 1~100, 실시간 조절
- **읽기 가이드 라인** — 현재 읽는 위치를 시각적으로 표시
- **섹션 라벨** — `// 섹션명` 문법으로 구간 구분
- **큐 힌트** — `[카메라를 보며]` 형식의 연출 메모
- **미러 모드** — 빔스플리터 장비용 좌우 반전
- **4가지 테마** — 다크, 스튜디오(녹색), 라이트, 고대비
- **시작 카운트다운** — 3초 카운트다운 후 재생
- **진행률·시간 표시** — 진행률, 경과/남은 시간
- **자동 저장** — localStorage에 대본·설정 보존
- **파일 가져오기/보내기** — .txt, .md 지원

## 단축키

| 키 | 동작 |
|---|---|
| `Space` | 재생 / 일시정지 |
| `↑` `↓` | 스크롤 속도 조절 |
| `E` | 편집 모드 |
| `M` | 미러 모드 |
| `F` | 전체화면 |
| `R` | 처음으로 |
| `?` | 단축키 도움말 |

## 실행

```bash
npm install
npm run dev
```

## 배포 (GitHub Pages)


## 기존 TelePrompter 대비 개선점

| 항목 | 기존 | TelePrompter Pro |
|---|---|---|
| 글씨 크기 | 4단계 프리셋 | 24~80px 연속 슬라이더 |
| 테마 | 단일 | 4가지 (스튜디오 포함) |
| 읽기 위치 | 고정 | 상단/중앙/하단 선택 |
| 진행 표시 | %만 | % + 경과/남은 시간 |
| 섹션 구분 | 없음 | `//` 라벨 지원 |
| 큐 힌트 | 없음 | `[힌트]` 문법 |
| 카운트다운 | 없음 | 3초 카운트다운 |
| 미러 모드 | 없음 | 지원 |
| 자동 저장 | 불명 | localStorage 자동 저장 |
| 컨트롤 | 항상 표시 | 재생 중 자동 숨김 |
